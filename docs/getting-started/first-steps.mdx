# First Steps

After logging in, start with these checks:

## 1. Confirm your role

Make sure your assigned role matches your responsibilities.

## 2. Review the dashboard

Check the main sections and current task overview.

## 3. Open a sample task

Read the task details, attachments, and status history.

## 4. Check permissions

Confirm that you can view, update, and close the items relevant to your role.
