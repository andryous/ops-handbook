# Ops Handbook

Welcome to the documentation demo for **Ops Handbook**.

This site is a simple example of an internal documentation hub built with Docusaurus.

## Purpose

The purpose of this documentation is to:

- help new users get started
- explain the main workflows
- provide basic troubleshooting guidance
- keep information structured and easy to find

## What you will find here

- **Getting Started** for access and first steps
- **Workflows** for core task handling
- **Support** for troubleshooting and common questions

## Scope

This is a demo project designed to show how technical documentation can be structured in a clear and maintainable way.
