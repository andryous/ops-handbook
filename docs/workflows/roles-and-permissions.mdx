# Roles and Permissions

This section explains the basic access model used in Ops Handbook.

## Administrator

Administrators can manage users, permissions, and workflow settings.

## Editor

Editors can create, update, and close tasks within their assigned scope.

## Viewer

Viewers can read task details and documentation, but cannot change records.

## Notes

Permissions should always match operational responsibility and business need.
