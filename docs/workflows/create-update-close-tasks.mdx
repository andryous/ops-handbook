# Create, Update, and Close Tasks

This section explains the basic task workflow.

## Create a task

Open the dashboard and select **New Task**.

## Update status

Move the task from **Open** to **In Progress** as work begins.

## Add supporting information

Attach notes, files, or comments to keep the task record complete.

## Close the task

Mark the task as **Done** and add final notes if required.
